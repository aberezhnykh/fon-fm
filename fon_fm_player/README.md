# FON.FM Flutter Player

Исходники Flutter-приложения под Windows с заделом на общий код для Android.

После установки Flutter SDK:
1. Выполнить `flutter create . --platforms=windows,android`
2. Выполнить `flutter pub get`
3. Запустить `flutter run -d windows`

`windows/` и `android/` здесь не сгенерированы, потому что в текущей среде отсутствует Flutter SDK.
